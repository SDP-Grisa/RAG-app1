Welcome to Semantic Chunking & QA App

This app lets you ingest .txt documents (semantic chunking + embeddings), ask questions (RAG with LLM), and check system health.

Use the sidebar to navigate. Start by uploading a document!

Upload & Ingest Document

Upload a .txt file to semantically chunk and store in the database via POST /ingest.

Ask a Question

Ask a question via POST /ask (retrieves chunks, generates answer with LLM).

Health Check

System diagnostics via GET /health.
